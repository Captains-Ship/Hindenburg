Installation

Installing Hindenburg

Via GitHub

Prerequisites
  • Git
  • Node.js
  • NPM (Comes installed with Node.js)
# Install yarn via npm
npm install -g yarn
# Copy source code to your computer
git clone https://github.com/Skeldjs/Hindenburg
cd Hindenburg
# Install dependencies and setup Hindenburg
yarn
yarn setup
# Use `yarn dev` instead to skip this step
yarn build
yarn start

yarn start will start Hindenburg, although this wil be closed when you close your terminal or end your SSH session.

Look into using something like pm2 if you want a solution to keep your server alive.

By default, Hindenburg will check for updates every time it is run. Set the checkForUpdates property in your config.json to false to disable this.

Or, you can enable auto-updating by setting the autoUpdate property to true.
